# Grand Education

Grand Education ushın React + TypeScript + Mantine based 5 betli frontend project.

## Start

```bash
npm install
npm run dev
```

## Routes

- /
- /courses
- /courses/:slug
- /about
- /results

## Main stack

- React
- TypeScript
- Mantine
- React Router DOM
- TanStack Query
- Zod
- Framer Motion
- React Leaflet
- Recharts
- i18n
- Storybook
- Vitest
- Bundle Analyzer
